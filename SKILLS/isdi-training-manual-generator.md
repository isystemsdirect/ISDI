# ISDI Training Manual Generator

## Description
Write and structure training manuals, onboarding documents, and employee handbooks.

## Instructions
1. Determine the specific training requirement (e.g., new employee onboarding, specific tool training like OVERSCITE, governance compliance).
2. Gather relevant source materials, standard operating procedures (SOPs), and canonical documentation.
3. Structure the training material logically, starting with objectives, core concepts, step-by-step instructions, and assessment criteria.
4. Draft the training manual, ensuring clear, accessible language while maintaining precise canonical terminology where required.
5. Include visual aids, diagrams, or references to system UI components where helpful.

## Output Format
- A comprehensive training manual or onboarding document (Markdown).
- Step-by-step guides and reference materials.
