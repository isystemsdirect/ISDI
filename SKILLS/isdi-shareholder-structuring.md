# ISDI Shareholder Structuring

## Description
Draft and document shareholder agreements, stock ledgers, and ownership classes.

## Instructions
1. Gather existing data on ISDI equity structure, outstanding shares, and ownership classes.
2. Draft or update shareholder agreements to reflect any new structuring or capital events.
3. Update and maintain the canonical stock ledger, ensuring all issuances and transfers are accurately recorded.
4. Define the rights and privileges of each ownership class clearly within the documentation.
5. Provide the structured documentation for corporate officer review.

## Output Format
- Shareholder agreement drafts.
- Updated capitalization table / stock ledger format.
