# ISDI Performance Analytics

## Description
Compile and report on ISDI's key performance indicators across different business units.

## Instructions
1. Aggregate performance data from various ISDI business units and operational systems.
2. Identify and track Key Performance Indicators (KPIs) relevant to each unit's objectives.
3. Analyze the data to identify trends, areas of operational efficiency, and potential bottlenecks.
4. Compile the findings into a structured performance analytics report.
5. Provide actionable recommendations based on the data analysis to improve overall performance.

## Output Format
- A detailed performance analytics report (Markdown).
- Data summaries and trend analysis charts (data tables).
