# OVERSCITE Preview Boundary

The OVERSCITE Preview is a publicly accessible demonstration of the OVERSCITE Global platform.

## Strict Rules
1. **Mock Data Only:** The preview must operate entirely on mock data located in `packages/demo-data`.
2. **No Real Customers:** Do not use real user accounts, inspection records, or property data.
3. **No Live Firestore:** The preview must not connect to operational Firestore collections.
4. **Visual Reuse:** Visual patterns and UI components can be reused from OVERSCITE Global, but they must be isolated and stripped of live backend connections.
